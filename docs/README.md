# Bulky Docs

This folder keeps planning, architecture, and release-reference material out of the product root so the app source stays easier to scan.

## Structure

- `architecture/`
  Production blueprints, capability matrices, and the codebase ownership map.
- `renovation/`
  Phase-by-phase renovation logs and status tracking.
- `releases/`
  Debug summaries and release-specific notes.
